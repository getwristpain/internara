# (Unstabled) Internara - Internship System

Internara is a comprehensive internship management system built with Laravel, designed specifically for schools and educational institutions to streamline and manage internship programs. This internal system allows schools to manage internship listings, track student applications, and facilitate communication between students and companies.

## Features

- Internship listing and applications
- Admin management dashboard
- User authentication
- Internship management specific to educational institutions
- Easy-to-use interface

## Installation

To set up Internara on your local machine, follow these steps:

### Prerequisites

Ensure you have the following installed:

- PHP >= 8.4
- Composer (Dependency Manager for PHP)
- MySQL or MariaDB
- Node.js >= 20.x
- npm (Node Package Manager)
- Git

### Steps to Install

1. Clone the repository:
    ```bash
    git clone https://github.com/getwristpain/internara.git
    cd internara
    ```

2. Install PHP dependencies:
    ```bash
    composer install
    ```

3. Install frontend dependencies:
    ```bash
    npm install
    ```

4. Run the installation command:
    ```bash
    php artisan app:install
    ```

5. Run the Laravel development server:
    ```bash
    php artisan serve
    ```

The application should now be available at [http://127.0.0.1:8000](http://127.0.0.1:8000).

## Usage

Once installed, you can log in as an admin or student to begin using the system. Students can apply for internships, while admins can manage the internship listings, monitor applications, and interact with companies. This system is designed for use within schools to help with internship management.
