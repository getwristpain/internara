# Contributing to Internara

Thank you for considering contributing to **Internara**! We welcome all types of contributions, from fixing bugs to adding new features.

## How to Contribute

### 1. Fork the Repository

To get started, fork this repository to your GitHub account.

### 2. Clone the Forked Repository

Clone the forked repository to your local machine:

```bash
git clone https://github.com/yourusername/internara.git
cd internara
```

### 3. Create a New Branch

Make sure to create a new branch for your changes:

```bash
git checkout -b feature-name
```

### 4. Make Changes and Commit

Make your changes to the code, and when you're ready, commit them:

```bash
git add .
git commit -m "Description of your changes"
```

### 5. Push Changes

Push your changes to your forked repository:

```bash
git push origin feature-name
```

### 6. Open a Pull Request

Once you've pushed your changes, open a pull request to the main repository.

### Code Style

Please ensure that your code adheres to the following:

- Follow PSR-12 for PHP code style.
- Ensure that JavaScript follows Airbnb’s JavaScript style guide.
- Keep code comments clear and concise.

### Testing

If you've made any code changes, please ensure that tests are written or existing tests are updated.

---

We look forward to your contributions! 🎉
